# Custom Fonts for PDF Generation

## Required Font Files

Place the following font files in this directory:

1. **Montserrat-Bold.ttf** - For "SKYLINE" main title
2. **Montserrat-Regular.ttf** - For "INSTITUTE OF TECHNOLOGY" subtitle
3. **Inter-Regular.ttf** - For address block and body text

## Where to Get Fonts

- **Montserrat**: https://fonts.google.com/specimen/Montserrat
- **Inter**: https://fonts.google.com/specimen/Inter

Download the font files and place them in this directory with the exact names above.

## Font Loading

The fonts are automatically registered in `src/components/pdf/PdfDocument.tsx` when the PDF component loads.
